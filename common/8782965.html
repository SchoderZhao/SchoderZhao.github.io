如果对单边操作,结合mybatis-plus可以 通过前端传入排序字段任意排序, 如果不适用mybatis-plus 自定义sql功能需要自己实现

  • 请求示例
 {
   "size": 10,   //每页条数  默认为10
   "current": 1, //页      默认为1
   "orders": [ //排序对象   
     {
       "column": "",   //需要排序的字段
       "asc": true     //排序方式  默认为ture asc
     }
   ],
   "data": {}     //需要传递的数据,在后端表现为泛型
 }
