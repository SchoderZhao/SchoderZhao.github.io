java公共组件 : 22.日志common-logging-starter

Created by 赵兴磊, last modified on 六月 22, 2022

引入依赖


implementation "com.itjcloud.common:common-logging-starter:${itjcloudVersion}"



配置

itjcloud:
  cloud:
    log:
      db-enable: false  #数据层日志 默认为true 根据自身项目情况调整
      controller-enable: false   #控制层日志 默认为true 根据自身项目情况调整
